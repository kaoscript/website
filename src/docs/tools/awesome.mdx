# awesome-kaoscript

## Color

- [@zokugun/lang.color](https://github.com/ZokugunKS/lang.color) - manipulate colors with ease (by default in RGB space)
- [@zokugun/lang.color.alvy](https://github.com/ZokugunKS/lang.color.alvy) - implements HSV/HSB, HSV, HWB and HSI spaces
- [@zokugun/lang.color.cie](https://github.com/ZokugunKS/lang.color.cie) - implements CIELAB, CIELUV, CIELCh, CIEXYZ, CIEYxy spaces
- [@zokugun/lang.color.xterm](https://github.com/ZokugunKS/lang.color.xterm) - let's manipulate terminal colors like any other colors

## Date and Time

- [@zokugun/lang.date](https://github.com/ZokugunKS/lang.date) - manipulate the built-in Date with ease
- [@zokugun/lang.timezone](https://github.com/ZokugunKS/lang.timezone) - brings the [tz database](https://en.wikipedia.org/wiki/Tz_database) into kaoscript

## Editor Plugins

- Visual Studio Code
  - [Kaoscript Language](https://marketplace.visualstudio.com/items?itemName=kaoscript.kaoscript-language) - the official VS Code extension for kaoscript

## Frameworks

- [@zokugun/lang](https://github.com/ZokugunKS/lang) - a logical framework - provides useful identical classes/functions on all supported engines.

## Internationalization

- [@zokugun/lang.i18n](https://github.com/ZokugunKS/lang.i18n) - provides root framework for internationalization libraries
- [@zokugun/lang.i18n.date](https://github.com/ZokugunKS/lang.i18n.date) - implements international dates and times

## Math

- [@zokugun/lang.math.matrix](https://github.com/ZokugunKS/lang.math.matrix) - implements common 3D matrix functions
- [@zokugun/lang.math.vector](https://github.com/ZokugunKS/lang.math.vector) - implements common 2D vector functions

## Template Engines

- [@zokugun/template](https://github.com/ZokugunKS/template) - fast and simple template engine

## Testing

- Code Coverage
  - [@kaoscript/coverage-istanbul](https://github.com/kaoscript/coverage-istanbul) - instrumentation for [Istanbul](https://github.com/gotwarlost/istanbul)
